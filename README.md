# MERN Interview Vault

> Senior-level interview prep notes for **MERN stack** developers (~6 YOE).
> JavaScript · TypeScript · DSA · React · Node/Express · MongoDB · Docker · AWS · System Design · Behavioral.

An [Obsidian](https://obsidian.md)-ready knowledge vault. **76 notes**, each following one format:

> **Question → Answer → Code (with & without built-in methods) → How / Why / Where / Scenario**

DSA notes give both the built-in-method solution **and** the from-scratch version (interviewers love "now do it without `.reverse()`").

---

## Use it

**In Obsidian (best):**
1. Install [Obsidian](https://obsidian.md).
2. `Open folder as vault` → select this folder.
3. Start at this README, then see [[00-How-To-Use-In-Obsidian]].
4. `[[wikilinks]]` connect notes; Graph view shows the whole map.

**On GitHub / any editor:** just browse the folders below — every file is plain Markdown.
